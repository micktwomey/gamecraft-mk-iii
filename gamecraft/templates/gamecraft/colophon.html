Colophon

This site is written using the following tools and frameworks:

  • Python (programming language)
  • Django (web framework)
  • PostgreSQL (data, hosted on Heroku)
  • MongoDB (static file storage, hosted on MongoSoup)
  • Redis (cache and ephemeral storage, hosted on Redis Cloud)
  • Bootstrap (CSS framework)
  • Less CSS (CSS pre-processor)

This site is hosted on Heroku and fronted by CloudFlare.

We use the following tools (many of which set cookies):

  • CloudFlare (caching and security)
  • TypeKit (fonts). In particular the excellent Museo and Museo Slab fonts are used.
  • New Relic (monitoring)
  • Sentry (error tracking)
  • Papertrail (logging)

For the most part this is a Django site running on Heroku. Mick just went a bit mad adding add-ons.

Code is hosted here on GitHub. This site is in large part an exercise in using pre-release Django 1.7 with Python 3.4 on Heroku. Essentially how to make things hard for ourselves.
